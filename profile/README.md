# Restful API 게시판 만들기

[Restful API 게시판](https://github.com/thdqudgnsToy/toy_board)
- **localhost/board** (POST) - 글작성
- **localhost/board** (PUT) - 글수정
- **localhost/board** (GET) - 글목록
- **localhost/board/:no** (GET) - 글조회
- **localhost/board/:no** (DELETE) - 글삭제
